Setup

Instruções completas de integração

1. Script para o <head> da pretzel

Substitui o script antigo por este. Captura click ID, gera Lead ID e atualiza o botão automaticamente:

<script> (function(){ var BASE = 'https://capi-track-production.up.railway.app'; function gp(n){var r=new RegExp('[?&]'+n+'=([^&#]*)').exec(location.href);return r?decodeURIComponent(r[1]):null;} var kwai_click_id = gp('clickid')||gp('kwai_click_id')||gp('kpf')||gp('click_id'); var utm_source = gp('utm_source'); var utm_campaign = gp('utm_campaign'); var utm_adset = gp('utm_adset')||gp('utm_content'); var utm_ad = gp('utm_ad')||gp('utm_term'); var session_id = 'sess_' + Math.random().toString(36).substring(2,10); // Cria o Lead ID no servidor fetch(BASE + '/kwai/lead/criar', { method: 'POST', headers: {'Content-Type':'application/json'}, body: JSON.stringify({ kwai_click_id: kwai_click_id, utm_source: utm_source, utm_campaign: utm_campaign, utm_adset: utm_adset, utm_ad: utm_ad, session_id: session_id }) }) .then(function(r){return r.json();}) .then(function(d){ if(!d.lead_id) return; var lid = d.lead_id; // Salva em cookie e localStorage var e=new Date();e.setDate(e.getDate()+7); document.cookie='kwai_lid='+lid+';expires='+e.toUTCString()+';path=/'; try{localStorage.setItem('kwai_lid',lid);}catch(x){} // Atualiza todos os links do botão com o lead_id document.querySelectorAll('a[href*="/kwai/ir"]').forEach(function(a){ try{var u=new URL(a.href);u.searchParams.set('lid',lid);a.href=u.toString();}catch(x){} }); }) .catch(function(){}); })(); </script>

⚠️ Substitui completamente o script antigo. O pixel do Kwai fica separado.

2. Link do botão

Vá em Números WhatsApp, selecione os números e clique em Gerar link. Cole no botão do Elementor.

3. Como o closer usa

O lead vai chegar no WhatsApp com a mensagem:
"Olá! Gostaria de saber mais sobre o tratamento. REF:A3K9F2"
O closer copia o código REF, abre Disparar Purchase, cola o REF, confirma o valor e dispara.
