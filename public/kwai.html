Setup

Como integrar na pretzel e usar o sistema

1. Script para a pretzel (WordPress)

Cole no <head> da pretzel via Insert Headers and Footers ou Elementor HTML:

<script> (function() { function gp(n){var r=new RegExp('[?&]'+n+'=([^&#]*)').exec(location.href);return r?decodeURIComponent(r[1]):null;} var cid=gp('clickid')||gp('kwai_click_id')||gp('kpf')||gp('click_id'); if(cid){ var e=new Date();e.setDate(e.getDate()+7); document.cookie='kwai_cid='+cid+';expires='+e.toUTCString()+';path=/'; try{localStorage.setItem('kwai_cid',cid);}catch(x){} } document.addEventListener('DOMContentLoaded',function(){ function gc(n){var m=document.cookie.match(new RegExp('(^| )'+n+'=([^;]+)'));return m?m[2]:null;} var id=gc('kwai_cid')||(function(){try{return localStorage.getItem('kwai_cid')}catch(x){return null}})(); document.querySelectorAll('a[href*="wa.me"],a[href*="whatsapp.com"]').forEach(function(a){ a.href='SEU_DOMINIO_RAILWAY/kwai/ir?cid='+(id||'')+'&campaign_id=glivia'; }); }); })(); </script>

⚠️ Substitua SEU_DOMINIO_RAILWAY pela URL do Railway onde o Infinity Track está rodando.

2. Como usar o closer

Quando fechar uma venda no WhatsApp:
1. Abra esta página
2. Vá em Disparar Purchase
3. Digite o número do cliente (ex: 6299988776)
4. Confira o lead e o valor
5. Clique em ⚡ Disparar Purchase
O evento vai direto para os 2 pixels do Kwai.
